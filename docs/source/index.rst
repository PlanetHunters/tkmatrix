Welcome to MATRIX documentation!
=============================================

.. image:: _static/matrix.jpg
   :width: 300px
   :align: center
   :height: 75px
   :alt: SHERLOCK PIPEline Logo

MATRIX ToolKit (ToolKit for Multi-phAse Transits Recovery from Injected eXoplanets) is a Python package prepared
for the astronomical community interested in running exoplanet transits injection and recovery tests on any
local light curve or an space-based mission one.
